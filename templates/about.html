About WeLearn

Project Overview

WeLearn is a web-based platform designed to support Chinese language learning. By leveraging real-world conversations from messaging platforms like WeChat, WeLearn provides a context-driven approach for learning vocabulary, pronunciation, and sentence structure in Chinese.

Key Features

  • Vocabulary Management: Automatically identify and store new words from conversations, along with pinyin and English translations.
  • Examples and Sentences: For each new word, WeLearn displays example sentences from real conversations, helping users understand usage in different contexts.
  • Difficulty Ratings: Users can rate the difficulty of each word, making it easier to track and prioritize learning.
  • Writing Practice: A handwriting canvas lets users practice writing Chinese characters without accidental page scrolling.
  • Progress Tracking: Track vocabulary learning progress with examples and ratings.

Technologies Used

This project is built with Flask, a Python web framework, and uses Jieba for Chinese text segmentation. The database management is handled through SQLAlchemy for secure and efficient data processing.
